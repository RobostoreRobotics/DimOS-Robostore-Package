# DimOS – Robostore Package

The files you need to follow the **RoboUniversity DimOS course** on a Unitree G1.

This package provides the course's custom DimOS blueprints, added on top of a stock
DimOS install. They are meant to be used **alongside the course**.

## How to use these

To understand how to install and use these files with DimOS, **follow the
RoboUniversity DimOS course** — it walks through every step, from connecting to your
robot to running these blueprints.
